Overview

The Sony DSC-HX100V is a feature-rich bridge camera built around a 1/2.3" Exmor R CMOS sensor offering approximately 16.2 effective megapixels. It pairs a Carl Zeiss Vario Sonnar T 30x optical zoom lens (4.8-144mm) with BIONZ processing to deliver versatile shooting from wide-angle to long-telephoto — including precision digital zoom, panorama modes, 3D capture and Full HD AVCHD movie recording.

Shooting Modes & Features

Available Modes

Intelligent Auto, Program Auto, Superior Auto, Manual Exposure, Aperture Priority, Shutter Priority, Intelligent Sweep Panorama, Background Defocus, Scene Selection (Landscape, Night, Beach, Snow, Fireworks, Gourmet, Pet, etc.), 3D modes, Advanced Sports, Anti Motion Blur, Handheld Twilight, High Sensitivity, Soft Skin, and many more.

Other Shooting Features

  • Face Detection (modes: Auto/Off/Child Priority/Adult Priority)
  • Smile Shutter
  • Burst Speed: Approx. 10.0 fps (10 shots)
  • Smart Zoom options & in-camera retouch (red-eye, unsharp masking, trimming)
Recording Media & Movie Features

Compatible Media

Memory Stick Duo (Still Image/Movies: Yes/No), SDHC, Memory Stick PRO Duo, Memory Stick PRO HG Duo, SD Memory Card, SDXC — and internal memory copy to recording media supported.

Movie Recording

  • AVCHD - 1920 x 1080 (17M FH / 24M FX / 28M PS)
  • AVC MP4 - 1280 x 720 (6M), VGA & other modes
  • Optical Zoom during movie recording: Yes
  • Wind Noise Reduction: Off/On
Playback & Interface

Playback features include slideshow with music (4 tunes), movie playback, 3D viewing mode, histogram indicator, automatic image rotation, and control for HDMI. Interface: Micro HDMI, DC IN terminal, Multi-use terminal (AV/USB), USB (MTP / Mass Storage / Auto).
